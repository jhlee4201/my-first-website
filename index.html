글의 제목을 설정하는 태그는 h1~h6(1>6)

HTML이란 무엇인가?

굵은 글씨 태그는 strong 밑줄 태그는 u(underline)

단락 하나를 설정하는 태그는 p(paragraph). 줄바꿈 태그는 br

이미지를 웹페이지에 삽입하는 태그는 img src="이미지 이름.확장자". img 뒤에오는 src(source)와 같은 태그가 바로 속성 태그

이미지의 크기를 설정하는 태그는 width="픽셀(ex.450) or __%(ex.50%)"

에디터의 오른쪽 아래를 보면 UTF-8이라는 확장자 명이 쓰여있음. 가끔 글씨가 깨지는 경우가 있는데 이는 나의 웹페이지를 해당 확장자(지금의 경우엔 UTF-8)로 열지 않았기 때문이다. 따라서 웹페이지를 열 때는 '이 윂페이지는 UTF-8로 열어야 한다'는 명령을 해줘야 하는데 이것이 meta charset="확장자 명"이다. charset는 character(문지)+set(규칙)의 합성어이다.

태그는 두가지 유형으로 나누어진다. 제목에 해당하는 head와 본문에 해당하는 body. 그리고 이 두가지 태그를 모두 감싸는 절대적인 하나의 태그가 html이다. 그리고 코드의 맨 위에는 '이 웹페이지는 html이다'를 설명해주는 코드 doctype html을 작성한다.

하이퍼링크를 거는 태그는 a(ahchor). 닻을 의미하는 a는 연결(link=hypertext)을 의미하기도 하며 html 태그의 제왕이자 정수이. a 태그 뒤에 속성 태그로 h(hypertext)ref(reference)와 해당 웹페이지의 주소를 입력한다. 새탭에서 열리게 하고 싶다면 target="_blank", 커서를 올려놓으면 정보가 간단하게 제공되는 툴팁을 만들고 싶다면 title="제목"과 같은 속성 태그를 추가로 입력한다.
